# Model Context Protocol (MCP)

## 定義

標準化代理與外部工具/資料源通訊的協定，定義工具發現、參數序列化、結果回傳、錯誤處理、流式回應等介面規範。類似 LSP (Language Server Protocol) 之於編輯器，MCP 旨在解決代理生態系工具互操作性問題。

## 為什麼重要

消除代理框架與工具提供者間的 N×M 適配成本，推動工具生態標準化。Google Gemini API Managed Agents 原生支援遠端 MCP，標誌協定進入生產級應用階段。
