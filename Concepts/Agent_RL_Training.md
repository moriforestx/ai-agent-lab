# Agent RL Training

## 定義

使用強化學習優化 Agent 策略：從試錯中學習最佳工具調用序列、多輪軌跡優化、任務完成度/效率/錯誤恢復綜合獎勵建模。

## 為什麼重要

Agent Lightning (Microsoft) 開源標誌從提示工程向策略學習范式轉移。Agent 行為不再凍結在 Prompt，而是通過 RL 持續改進。
