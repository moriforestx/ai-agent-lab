# Object Detection Transformer

## 定義

基於 Vision Transformer (ViT) 架構的物件偵測模型，透過自注意力機制全域建模圖像特徵，取代或輔助 CNN 骨幹。

## 為什麼重要

Transformer 在大規模預訓練後展現優越的特徵表達能力，RT-DETR 等模型實現端到端即時偵測，無需 NMS 後處理。
