# Token-and-Duration Transducer (TDT)

## 定義

語音辨識架構，聯合建模 Token 序列與持續時間，天生支援流式推理，無需外部對齊模組。

## 為什麼重要

結合 RNN-T 的流式優勢與注意力機制的建模能力，實現高精度即時語音辨識。
