# Long Context LLM

## 定義

原生支援長上下文窗口（32k+ tokens）的大語言模型，通過擴展位置編碼、稀疏注意力、環形注意力等技術突破標準 Transformer 的上下文長度限制。

## 為什麼重要

長上下文能力對於代碼庫分析、長文檔問答、多輪對話記憶、RAG 大規模檢索等場景至關重要。
