# Agentic Workflow Platform

## 定義

支援 Agent 工作流全生命週期的平台：視覺化編排、內建 RAG、多模型管理、觀測性、RBAC、多租戶、本地/雲部署。代表：Dify、Langflow。

## 為什麼重要

解決「Demo 易、產品難」。企業級 Agent 平台標桿，提供從原型到生產的完整工具鏈。
