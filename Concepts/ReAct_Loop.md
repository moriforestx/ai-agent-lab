# ReAct Loop (Reasoning + Acting)

## 定義

代理核心推理循環：Thought (思考) → Action (行動) → Observation (觀察) → ... 迭代直到得出最終答案。將推理與行動統一在單一提示流程中，支援多步驟工具使用與自我修正。

## 為什麼重要

奠定現代代理系統的基礎推理範式，LangGraph、CrewAI、AutoGen 等框架均以 ReAct 為核心循環實現。
