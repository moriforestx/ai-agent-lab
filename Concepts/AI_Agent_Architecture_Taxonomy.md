# AI Agent Architecture Taxonomy

## 定義

AI Agent 系統架構標準化分類：三大支柱 —— (1) 審議推理：CoT 分解、自我反思驗證、約束感知決策 (2) 規劃控制：反應式 → 分層多步規劃器 (3) 工具調用環境交互：檢索、代碼執行、API、多模態感知。模塊化工具路由 (MRKL) 分離語言理解與專用工具。

## 為什麼重要

AI Agent Systems (arXiv:2601.01743) 建立領域首個系統性架構分類學，統一術語，指導研究與工程。
