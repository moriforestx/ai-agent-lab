# Long Context RAG Synergy

## 定義

長上下文 LLM (2M+ tokens) 與 RAG 非對立，而是協同：長上下文擴展 RAG 設計空間，RAG 為長上下文提供相關性過濾與知識更新機制。

## 為什麼重要

打破「長上下文取代 RAG」迷思。Long-Context LLMs Meet RAG (arXiv:2410.05983) 實證：隨檢索段落增加，生成質量先升後降；提出三種魯棒性方案：訓練免費重排序、隱式微調 (難負樣本)、顯式微調 (中間推理)。
