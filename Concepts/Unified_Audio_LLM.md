# Unified Audio LLM

## 定義

統一音頻理解 (ASR、音頻理解、語音對話) 與生成 (TTS、音樂生成、聲音克隆) 的大模型，共享離散音頻 Token 表示空間。

## 為什麼重要

Step-Audio 2 (arXiv:2507.16632) 潛在音頻編碼器 + 推理導向 RL，端到端語音對話，ASR 多語言 SOTA，130B 參數規模。InstructAudio (arXiv:2511.18487) 自然語言指令統一語音音樂生成。音頻 LLM 邁向產業級。
