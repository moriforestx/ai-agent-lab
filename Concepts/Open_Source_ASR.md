# Open Source ASR

## 定義

完全開源（模型權重、訓練代碼、數據集）的自動語音識別模型，可私有化部署、商業使用、微調適配。2026 年 Parakeet TDT 1.1B、Granite 4.0 1B Speech 在 LibriSpeech test-clean 達 1.4% WER，正式達到商用級水準。

## 為什麼重要

打破商業 API (Google Cloud STT、Azure Speech、AWS Transcribe) 的精度壟斷，企業可零授權費自建語音管線，資料不出本地。
