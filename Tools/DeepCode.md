# DeepCode

## 這是什麼

端到端代碼生成 Agent：從需求到可部署代碼。港大 HKUDS 開源。

## 主要功能

- 需求理解：自然語言 → 技術規範
- 架構設計：模塊劃分、技術選型、數據庫設計
- 代碼生成：項目結構、業務邏輯、測試、部署配置
- 迭代優化：基於反饋修改、重構
- 多語言支援

## 為什麼重要

編碼 Agent 進化方向：從代碼補全 → 端到端軟體工程自動化。
