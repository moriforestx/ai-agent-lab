# Awesome AI DevTools

## 這是什麼

AI 原生 IDE 與開發工具精選列表：Cursor、Replit、Google Antigravity (Agent-first IDE)、Theia IDE、GitHub Copilot、Amazon Q Developer 等分類匯總。GitHub: jamesmurdza/awesome-ai-devtools

## 主要功能

- AI-Native IDEs: Cursor, Replit, Antigravity, Theia IDE
- IDE Extensions: GitHub Copilot, Amazon Q Developer, Tabnine
- CLI Agents: Claude Code, OpenCode, Gemini CLI, OpenClaw CLI
- Browser Automation: Playwright, Browser-Use, Stagehand
- Code Generation: DeepCode, v0, Bolt.new
