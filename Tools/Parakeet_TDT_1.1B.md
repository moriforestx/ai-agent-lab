# Parakeet TDT 1.1B

## 這是什麼

NVIDIA 開源的 Token-and-Duration Transducer (TDT) 語音辨識模型，1.1B 參數，在 LibriSpeech test-clean 達成 1.4% WER，並列開源模型榜首。

## 主要功能

- 推理速度快：TDT 架構天生流式，支援即時轉錄
- 多語言支援：可微調擴展至其他語言
- 開源授權：Apache 2.0，商業可用
- 整合簡單：NeMo 工具包原生支援

## 為什麼重要

重新定義開源 ASR 精度天花板，接近商用級服務水準，且完全可控、可私有化部署。

## 可能影響我

語音相關專案可直接採用，避免商用 API 費用與隱私風險。
