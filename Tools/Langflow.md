# Langflow

## 這是什麼

低代碼平台：視覺化設計、部署 AI Agent 與 RAG 工作流。基於 LangChain 構建。

## 主要功能

- 拖拽式工作流編輯器
- 內建組件：LLM、向量庫、檢索器、工具、Agent
- 即時預覽、調試
- 一鍵部署 API/聊天界面
- Python 代碼導出
- 多用戶、項目隔離

## 為什麼重要

降低 Agent 開發門檻。非技術人員可參與原型驗證。快速迭代 RAG/Agent 管線。

## 可能影響我

- 原型開發加速
- 非技術協作場景
- 工作流導出為代碼接入生產
