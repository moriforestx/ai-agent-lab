# Dify

## 這是什麼

生產級 Agentic 工作流平台。全鏈路工具鏈：構建、部署、管理 AI 應用。

## 主要功能

- 視覺化工作流編排器
- 內建 RAG 管線管理 (文檔解析、分塊、嵌入、檢索、重排)
- 多模型供應商支援 (OpenAI, Anthropic, 開源模型)
- 使用量監控、成本控制、審計日誌
- 本地部署 (Docker/K8s) + 雲端 SaaS
- 插件系統、工具市場
- RBAC、多租戶

## 為什麼重要

企業級 Agent 平台標桿。解決「Demo 易、產品難」。

## 可能影響我

- 生產部署架構參考
- 平台化能力建設藍圖
- 多租戶、觀測性、治理最佳實踐
