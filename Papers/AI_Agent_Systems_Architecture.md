# AI Agent Systems: Architectures, Applications, and Evaluation

## 基本資訊

- 類別：AI Agents
- 日期：2026-01
- 新鮮度：0–3 個月
- Source：arXiv:2601.01743v1
- Score：5

## 摘要

系統性綜述 AI Agent 架構：(i) 審議推理：CoT 分解、自我反思驗證、約束感知決策 (ii) 規劃控制：從反應式策略到分層多步規劃器 (iii) 工具調用環境交互：檢索、代碼執行、API、多模態感知。模塊化工具路由 (MRKL 風格) 將語言理解與專用工具分離，強制結構化接口便於審計。任務範圍從寫作助手擴展至工作流自動化：編碼 Agent 端到端解決 Issue、Web Agent 真實環境操作、企業助手多步操作與策略約束。

## 核心重點

- 三大架構支柱：推理、規劃、工具調用
- 模塊化工具路由改善治理與可審計性
- 應用演進：寫作 → 工作流自動化 (編碼、Web、企業)
- 基準與評估體系發展

## 為什麼重要

AI Agent 領域首個系統性架構綜述，確立術語與分類標準，指導後續研究與工程實踐。
