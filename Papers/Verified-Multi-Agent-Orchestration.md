# Verified Multi-Agent Orchestration

## 基本資訊
- 類別：AI Agents

- Source：https://arxiv.org/abs/2603.11445

## 摘要
ICLR 2026 MALGAI Workshop 論文，提出經驗證的多代理協調框架：Plan-Execute-Verify-Replan (PEVR) 四階段循環。

## 核心重點
- 四階段循環：規劃 → 執行 → 驗證 → 重新規劃
- 形式化驗證：每步執行結果經自動驗證器檢查

## 為什麼重要
多代理系統可靠性長期依賴啟發式設計，PEVR 框架引入形式化驗證。

## 可能影響我
OpenClaw 多代理協調技能可參考 PEVR 框架。
